# Obsidian Plugins for DevOps

## Что поставить сейчас

### Dataview

Для автоматических таблиц и списков по заметкам.

Зачем нам:

- собирать список всех ошибок;
- собирать список всех команд;
- показывать проекты по статусу;
- делать обзор прогресса.

Установка:

```text
Settings -> Community plugins -> Browse -> Dataview -> Install -> Enable
```

### Tasks

Для задач внутри заметок.

Зачем нам:

- вести todo по обучению;
- отмечать практические задания;
- собирать незавершенные задачи из всего vault.

Установка:

```text
Settings -> Community plugins -> Browse -> Tasks -> Install -> Enable
```

### Templater

Для умных шаблонов.

Зачем нам:

- быстрее создавать дневные занятия;
- автоматически вставлять дату;
- делать шаблоны ошибок, команд, проектов.

Установка:

```text
Settings -> Community plugins -> Browse -> Templater -> Install -> Enable
```

## Что поставить позже

### QuickAdd

Для быстрых действий одной командой.

Зачем нам позже:

- быстро создать новую ошибку;
- быстро создать заметку команды;
- быстро добавить итог занятия.

Установка:

```text
Settings -> Community plugins -> Browse -> QuickAdd -> Install -> Enable
```

### Spaced Repetition

Для DevOps не обязательно, но можно использовать для запоминания команд.

Пример карточки:

```text
Что делает chmod +x?::Добавляет файлу право на выполнение.
```

## Минимальный набор для DevOps-Linux

```text
Dataview
Tasks
Templater
```
