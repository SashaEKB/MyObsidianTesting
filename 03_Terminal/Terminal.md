# Terminal

## Базовые навыки

- Копирование в терминал: `Ctrl+Shift+V`.
- Копирование из терминального приложения: `Shift` + выделение, потом `Ctrl+Shift+C`.
- Очистить набранную строку: `Ctrl+U`.
- Остановить процесс в терминале: `Ctrl+C`.

## Команды для разбора

- `cd`
- `ls`
- `pwd`
- `chmod`
- `groups`
- `which`
- `systemctl`
- `journalctl`
- `nvidia-smi`
