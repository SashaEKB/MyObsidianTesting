# Команды которые я уже использовал

## Система

```bash
uname -r
```

Показывает версию текущего ядра Linux.

```bash
groups
```

Показывает группы текущего пользователя.

```bash
which obsidian
```

Показывает путь к исполняемому файлу программы.

## NVIDIA

```bash
nvidia-smi
```

Показывает состояние видеокарты и процессы на GPU.

## Python

```bash
python3 -m venv .venv
```

Создать виртуальное окружение.

```bash
source .venv/bin/activate
```

Активировать виртуальное окружение.

```bash
python -m pip install -r requirements.txt
```

Установить зависимости проекта.

## Файлы и запуск

```bash
chmod +x file.sh
```

Разрешить запуск файла.

```bash
mkdir -p ~/.npm-global
```

Создать пользовательскую папку для глобальных npm-пакетов.

```bash
npm config set prefix ~/.npm-global
```

Настроить `npm`, чтобы глобальные пакеты ставились в домашнюю директорию, а не в `/usr/local`.

```bash
npm config get prefix
```

Показать, куда `npm` ставит глобальные пакеты.

```bash
echo $SHELL
```

Показать текущий shell: например, `/bin/bash` или `/bin/zsh`.

```bash
echo 'export PATH="$HOME/.npm-global/bin:$PATH"' >> ~/.bashrc
```

Добавить пользовательскую npm-папку с командами в `PATH` для `bash`.

```bash
source ~/.bashrc
```

Перечитать настройки `bash` без перезапуска терминала.

## npm и Claude Code

```bash
npm install -g @anthropic-ai/claude-code
```

Установить Claude Code как глобальную npm-команду.

```bash
claude --version
```

Проверить, что Claude Code установлен и доступен в терминале.

```bash
claude
```

Запустить Claude Code.
