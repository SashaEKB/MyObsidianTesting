## Быстрый Конспект

> [!summary] Запомнить
> Репозиторий на GitHub удаляется через сайт GitHub, а не через `git push` или локальные команды.

**Главное:**

- Путь: repository -> `Settings` -> `Danger Zone` -> `Delete this repository`.
- GitHub попросит подтверждение, чтобы защитить от случайного удаления.
- Локальная папка на компьютере после удаления GitHub repository остаётся.
- Если локальная копия осталась, проект можно снова залить на GitHub.

**Практика после видео:** удалить только тестовый repository и проверить, что локальная папка не исчезла.

## Идеи Без Воды

> [!ai] AI extraction
> Главные мысли, которые реально стоит оставить в Obsidian.

- Удаление репозитория на GitHub делается не через Git-команду, а через настройки репозитория на сайте GitHub.
- Опасные действия находятся в `Settings` -> `Danger Zone`.
- GitHub специально просит подтверждение и ввод названия репозитория, чтобы защитить от случайного удаления.
- Удаление репозитория на GitHub не удаляет локальную папку проекта на компьютере.
- Если локальная папка осталась, код можно снова опубликовать в новый репозиторий.
- Если проект важный, сначала оставить локальную копию или backup, а уже потом удалять GitHub repository.
- Если это тестовый репозиторий, можно удалить и GitHub repository, и локальную папку, чтобы не путаться.

## Безопасный Алгоритм Удаления

1. Открыть repository на GitHub.
2. Проверить название repository.
3. Открыть `Settings`.
4. Пролистать вниз до `Danger Zone`.
5. Нажать `Delete this repository`.
6. Прочитать предупреждение.
7. Ввести подтверждение, которое просит GitHub.
8. Удалить repository.
9. Проверить локальную папку на компьютере.
10. Удалять локальную папку только если она точно больше не нужна.

## Частая Ошибка Новичка

> [!warning] Важно
> `git push`, `git remote remove origin` и удаление папки `.git` не удаляют репозиторий на GitHub. Они меняют только локальную связь или локальный Git. Сам GitHub repository удаляется через сайт GitHub.

## Термины

| Термин | Простое объяснение |
| --- | --- |
| GitHub repository | Проект на GitHub: файлы, commits, branches, issues, settings. |
| Settings | Раздел настроек конкретного repository на GitHub. |
| Danger Zone | Раздел GitHub с опасными действиями: удалить repository, архивировать, изменить видимость. |
| Delete this repository | Кнопка удаления репозитория на GitHub. |
| Local copy | Папка проекта на твоём компьютере. Она не исчезает автоматически после удаления GitHub repository. |
| Backup | Резервная копия, чтобы можно было восстановить данные. |
| Remote | Ссылка локального Git на удалённый repository, например `origin`. |
