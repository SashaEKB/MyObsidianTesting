# Карта изучения Git

Эта папка — практика Git с ИИ. Здесь короткие заметки, чтобы быстро повторить команды и понять смысл.

## Как учить

1. Сначала читай смысл команды.
2. Потом вводи команду руками, не копируй.
3. После каждой команды смотри результат через `git status`.
4. Если ошибка — не пугайся, Git обычно прямо пишет, что нужно сделать.
