# Что такое Git

Git — система контроля версий.

Он помогает:

- сохранять историю изменений;
- возвращаться к старым версиям;
- видеть, что именно изменилось;
- работать с ветками;
- отправлять код на GitHub или GitLab.

## Простая аналогия

Коммит — это сохраненная точка в истории проекта.

Пример:

```text
Версия 1: создал файл
Версия 2: добавил текст
Версия 3: исправил ошибку
```

Git позволяет перейти к любой такой точке и посмотреть, что было изменено.

## Важно запомнить

Git следит не за всей системой Linux, а только за папкой, где был выполнен `git init`.
