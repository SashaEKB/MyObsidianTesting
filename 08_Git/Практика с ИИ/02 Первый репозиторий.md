# Первый репозиторий

Репозиторий — это папка проекта, в которой Git хранит историю изменений.

## Создать папку для практики

```bash
mkdir git-practice
```

`mkdir` создает новую папку.

## Перейти в папку

```bash
cd git-practice
```

`cd` меняет текущую папку в терминале.

## Включить Git в этой папке

```bash
git init
```

`git init` создает скрытую папку `.git`. В ней Git хранит историю проекта.

## Проверить состояние

```bash
git status
```

`git status` показывает, что происходит в репозитории сейчас.

## Создать файл

```bash
nano hello.txt
```

`nano` открывает текстовый редактор в терминале.

`hello.txt` — имя файла.

## Настроить автора коммитов

```bash
git config --global user.name "Kraz"
git config --global user.email "your-email@example.com"
```

`git config` меняет настройки Git.

`--global` значит: настройка действует для всех Git-проектов этого пользователя.

`user.name` и `user.email` записываются в каждый коммит.
