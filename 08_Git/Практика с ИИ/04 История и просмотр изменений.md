# История и просмотр изменений

Эти команды помогают понять, что уже было сделано.

## Посмотреть историю

```bash
git log
```

Показывает полную историю коммитов: автора, дату, номер коммита и сообщение.

## Короткая история

```bash
git log --oneline
```

`--oneline` показывает каждый коммит в одну строку.

Удобно, когда коммитов много.

## Посмотреть изменения до коммита

```bash
git diff
```

Показывает, что изменилось в файлах, но еще не подготовлено через `git add`.

## Посмотреть подготовленные изменения

```bash
git diff --staged
```

Показывает изменения, которые уже добавлены через `git add`, но еще не закоммичены.

## Посмотреть конкретный коммит

```bash
git show <commit_id>
```

`git show` показывает детали одного коммита.

`<commit_id>` — номер коммита из `git log --oneline`.
