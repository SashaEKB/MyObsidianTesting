# Шпаргалка команд

## Старт

```bash
git init
```

Создать Git-репозиторий в текущей папке.

```bash
git clone <url>
```

Скачать готовый репозиторий.

## Проверка

```bash
git status
```

Посмотреть состояние проекта.

```bash
git log --oneline
```

Посмотреть короткую историю коммитов.

```bash
git diff
```

Посмотреть изменения до `git add`.

## Сохранение

```bash
git add <file>
```

Подготовить один файл к коммиту.

```bash
git add .
```

Подготовить все изменения.

```bash
git commit -m "message"
```

Сохранить подготовленные изменения в историю.

## Ветки

```bash
git branch
```

Показать ветки.

```bash
git switch <branch>
```

Перейти на ветку.

```bash
git switch -c <branch>
```

Создать ветку и перейти на нее.

## GitHub

```bash
git remote -v
```

Показать удаленные репозитории.

```bash
git remote set-url origin git@github.com:SashaEKB/testing-GIT.git
```

Переключить remote на SSH.

```bash
ssh-keygen -t ed25519 -C "your-email@example.com"
```

Создать SSH-ключ.

```bash
ssh -T git@github.com
```

Проверить SSH-доступ к GitHub.

```bash
git push
```

Отправить коммиты.

```bash
git pull
```

Получить изменения.
